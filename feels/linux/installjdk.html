java -version 检查是否存在。
        删除自带的openjdk：rpm -e nodeps jdk版本
        Linux安装jdk：
            1.在主机下载好Linux版本的jdk，传输到Linux指定文件夹
            2.解压文件到目录，tar -zxvf jdk文件名 目录
            3.配置环境变量 vi /etc/profile
                export JAVA_HOME=/usr/lib/jvm/jdk-13.0.1
                export JRE_HOME=${JAVA_HOIME}/jre
                export CLASSPATH=.:${JAVA_HOME}/lib:${JRE_HOME}/lib
                export PATH=${JAVA_HOME}/bin:$PATH
            4.java -version
