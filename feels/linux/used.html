        通过 /？？？  查询文件
        find /(在指定目录) -name filename  通过文件名称找名字  (-ls) 查看详细信息，权限等
        find /(在指定目录) -size 127b(b是默认单位，意思是块大小，一块大概512个字节、c 代表多少个字符，w代表多少个字)
        如果你指定 --size 200w ，你将会找到 200 个“字word”的文件——以“双字节增量”为单位的字，而不是“我们互相谈论的那些事情”中的单词。
        你还可以通过以千字节（k）、兆字节（M）和千兆字节（G）为单位提供大小来查找文件。（LCTT 译注：乃至还有 T、P）
        cat /proc/version 查看系统版本信息
        查看centos版本：cat /etc/redhat-release
        vim /etc/vimrc  在最后加入set nu ,显示行数
        ps命令，process status 用来列出系统中当前运行的那些进程
        firewall-cmd --state 查看状态
        firewall-cmd --list-all 所有开放的端口
        netstat -antpl | grep 名称/端口 查看端口使用情况
        firewall-cmd --permanent --zone=public --add-port=8080-8081/tcp //永久
        firewall-cmd --zone=public --add-port=8080-8081/tcp //临时
        irewall-cmd --permanent --zone=public --list-services //服务空格隔开 例如 dhcpv6-client https ss
        firewall-cmd --permanent --zone=public --list-ports //端口空格隔开 例如 8080-8081/tcp 8388/tcp 80/tcp
        关闭
        systemctl stop firewalld.service
        禁止开机自启
        systemctl disable firewalld.service
